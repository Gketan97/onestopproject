BEFORE YOU START

What Swiggy measures — and how

THE METRICS

ORDERS

Net completed orders — gross minus cancellations. At ~2.4M orders/day, a 12% WoW drop is ~290K orders missing.

WoW

Week over Week — this Tuesday vs last Tuesday. Day-of-week matching removes weekend effects from the comparison.

USERS

New = first order ever. Returning = ordered in last 30 days. Resurrected = 30+ day lapse then came back. Each type has a different owner and a different fix.

THE THREE USER TYPES — VISUALISED

Each type has a different owner and a different fix. A 15% drop in returning users is a completely different problem from a 15% drop in new users — even though both show the same headline number.
